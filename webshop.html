Webshop

Tijdens mijn opleiding aan Karel de Grote Hogeschool heb ik gewerkt aan het project Webshop KDG, een leerzame en uitdagende opdracht waarin ik de basisprincipes van HTML, CSS, en JavaScript onder de knie kreeg. In dit project lag de focus op het bouwen van een functionele webshop vanaf de grond. Het begon met het structureren van de pagina's met HTML, waarbij ik leerde hoe je semantische elementen gebruikt om een duidelijke en toegankelijke website op te zetten. Vervolgens gebruikte ik CSS om de webshop visueel aantrekkelijk te maken. Ik experimenteerde met stijlen, kleuren, lettertypen en layouts om de pagina's professioneel en gebruiksvriendelijk te laten ogen. De grootste uitdaging was het toepassen van JavaScript om interactieve elementen toe te voegen. Dit omvatte onder andere: Het implementeren van eenvoudige validatie voor formulieren. Het verwerken van gebruikersacties, zoals het toevoegen van producten aan de winkelwagen en het aanpassen van aantallen.

Het project gaf me niet alleen inzicht in hoe een webshop technisch in elkaar steekt, maar ook in de samenwerking tussen de verschillende lagen van een website (structuur, stijl en functionaliteit). Dankzij dit project heb ik een stevige basis gelegd in webontwikkeling en kreeg ik een voorproefje van hoe creatief en oplossingsgericht werken in de praktijk eruitziet. Het resultaat was een eenvoudige maar goed functionerende webshop die ik met trots als mijlpaal in mijn leerproces beschouw.

Dingen die ik tijdens dit project heb geleerd
HTML HTMl syntax, juiste tags gebruiken en Conventies.
CSS Het correcte gebruik van css ids en classes.
JavaScript Het gebruik van DOM en eventlisteners.
JavaScript Het gebruik van JavaScript voor controles op formulieren en opslaan van deze data.
