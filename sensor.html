Data Science

In dit project ontwikkelde ik een eenvoudig Internet of Things (IoT)-systeem met een ESP32-microcontroller. Het apparaat verzamelde sensordata en stuurde deze naar een webinterface, waar real-time monitoring mogelijk was. Functionaliteit van het Project

Ik gebruikte een ESP32 om data te verzamelen van een temperatuur- en vochtsensor (DHT11). De ESP32 communiceerde via Wi-Fi met een server.

Met behulp van C/C++ en de Arduino IDE schreef ik firmware die de sensordata uitleest en periodiek verstuurt naar een webserver via een HTTP POST-request.

Ik maakte een eenvoudige webpagina die sensordata in real-time weergeeft. Data werd opgeslagen in een database (via MySQL) en weergegeven met grafieken die zijn gemaakt met Chart.js.

Het systeem maakte gebruik van een beveiligde Wi-Fi-verbinding en versleutelde dataoverdracht via HTTPS om te voorkomen dat gegevens tijdens de transmissie onderschept werden.

Dingen die ik tijdens dit project heb geleerd
IOT apparaten Inzicht in de werking van IoT-apparaten, van hardware tot software-integratie.
Beveiligingsprincipes Ik leerde hoe microcontrollers worden geprogrammeerd en hoe data effectief wordt verwerkt en beveiligd in een netwerkomgeving.
